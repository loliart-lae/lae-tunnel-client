# lae-tunnel-client
Light App Engine Tunnel Client

支持平台：`Windows` `Linux` `macOS`

## 如何使用

请注意：或许你需要自备 frpc 程序, 你可以在 `config.yml` 修改启动命令

### Windows
1. 在 Releases 下载最新的版本.
2. (可选) 修改 `config.yml`
3. 运行 `lae-tunnel.exe`
4. (可选) 使用启动参数运行

### Linux
1. 确保安装 `Python3`
2. 在 Releases 下载最新的版本.
3. (可选) 修改 `config.yml`
4. 执行 `python lae-tunnel.py`
5. (可选) 使用启动参数运行

### macOS
同 Linux.
